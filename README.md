# AI-Career-Hub

AI Career Hub is a modern MERN Stack web application designed to help students and fresh graduates accelerate their careers using Artificial Intelligence.

## Features

- AI Resume Builder
- AI Resume Analyzer
- AI Mock Interview
- AI Career Guidance
- Job Recommendations
- JWT Authentication
- Responsive Design
- Modern UI
- AI Integrated Services

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Future Improvements

- OpenAI/Gemini Integration
- Resume PDF Generator
- AI Chatbot
- Email Notifications
- Admin Dashboard
